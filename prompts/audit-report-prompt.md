# Audit Report Prompt

Create a polished audit report for a landing page using the provided extracted data and scoring results.

Required sections:

1. Hero with page name, audited URL, overall score, and one-line verdict.
2. Score cards for SEO, GEO, AEO, CRO, and authority.
3. Biggest missed opportunity.
4. Evidence summary.
5. Priority fixes split into Fix now, Improve next, and Build over time.
6. Suggested SEO improvements.
7. Suggested GEO/AEO improvements.
8. Rewritten landing page section.
9. 30-day action plan.

Every fix must include:

- Issue.
- Why it matters.
- Evidence.
- Recommended fix.
- Impact.
- Effort.

Keep the report suitable to send to a prospect.
