# Landing Page Rewrite Prompt

Rewrite the hero and first page section for the audited landing page.

Use only evidence from the page and conservative inferred positioning from the title, H1, meta description, headings, CTAs, and visible body copy.

Return:

- Headline.
- Subheadline.
- Three concise bullets.
- Primary CTA.
- Trust/proof row.

The copy should be clear, commercial, and conversion focused. Do not invent proof. If proof is missing, use placeholder proof categories such as reviews, case studies, accreditations, or client results.
