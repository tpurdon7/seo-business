# Humanizer Skill Placeholder

Use this file for report and rewrite style rules.

Voice:

- Direct.
- Concise.
- Commercial.
- Specific.
- Plain spoken.

Avoid:

- Generic AI marketing language.
- Inflated claims.
- Fake certainty.
- Invented proof.
- Em dashes.
- Phrases like unlock, elevate, seamless, game-changing, in today's digital landscape, crucial, pivotal, and let's dive in.

When rewriting landing page copy, keep the meaning grounded in extracted page evidence. If a detail is missing, recommend adding it instead of pretending it exists.
